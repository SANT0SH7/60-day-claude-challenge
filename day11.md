# 🤖 ATS Resume Optimizer — Day 11 of 60-Day AI Challenge

---

## 📌 What This Project Does

ATS (Applicant Tracking Systems) act as the first screening layer for most companies. Before a human recruiter ever sees your resume, an algorithm scans it for keyword matches, structure, and relevance.

This tool helps candidates beat that filter — **ethically and accurately** — by:

- Scoring resume-to-JD alignment (ATS Match Score out of 100)
- Identifying missing keywords and skill gaps
- Rewriting the resume using JD language naturally
- Never inventing experience, metrics, or certifications

## ⚙️ Features

- **ATS Match Score** — percentage score with visual ring indicator
- **Gap Analysis** — missing keywords, missing skills, and matched skills
- **Optimized Resume Output** — fully rewritten, copy-paste ready
- **Plain text export** — paste into Word, Google Docs, FlowCV, or Canva
- **Dark mode support** — works across all themes

---

## 🛠️ Tech Stack

- HTML / CSS / JavaScript (single-file, no framework)
- Claude AI (Anthropic) for resume analysis and rewriting
- Tabler Icons for UI

---

## 🚀 How to Use

1. Clone the repo or open `index.html` directly in a browser
2. Paste your existing resume text
3. Paste the target job description
4. Click **Analyze** — get your ATS score, gap analysis, and rewritten resume
5. Copy the optimized resume and paste it into your preferred editor
