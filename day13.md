## 🎯 What I Did Today

Used **Claude + Claude Connectors** to build a complete AI-powered job search pipeline — from resume analysis to opportunity matching, skill gap analysis, and market demand insights.

---

## 🔧 How It Works

The workflow followed 3 structured prompts:

### Prompt 1 — Professional Profile
Fed Claude my resume (Santosh Kumar, B.Tech student, Full-Stack & DSA enthusiast).  
Claude extracted: skills, certifications, education, achievements, and experience level.

### Prompt 2 — Job Search Criteria
Defined target roles, preferred locations (open to all — remote/hybrid/onsite), and experience level (0–2 years / fresher).

### Prompt 3 — Job Discovery & Analysis
Claude searched for real, matching job opportunities and returned:
- 8 matched roles with fit scores
- Direct application links
- Skill gap analysis
- Market demand insights
- Interview preparation recommendations

## ⚠️ Skill Gap Analysis

1. **No database experience (SQL / MongoDB)** — Critical gap for full-stack roles. Even a small CRUD project would make a big difference.
2. **No backend / Node.js experience** — Most full-stack roles require Express + Node. Build one REST API project to close this gap.
3. **No public portfolio or deployed projects** — Resume has no Projects section. GitHub repos with READMEs are the #1 thing recruiters check for freshers.

---

## 💡 Surprising Insight

> Despite having **6 Coursera certifications** and skills in React, Python, Java, C++, and DSA — the biggest blocker is the **absence of deployed projects on GitHub**.  
> Recruiters don't just want skills. They want **proof**.

---

## ✅ Action Plan (Claude's Recommendations)

1. Build 2–3 real projects and push them to GitHub (React app, Python + API, MERN mini-app)
2. Add a "Projects" section to the resume with tech stack + GitHub links
3. Learn Node.js + Express basics (2–3 weeks) to unlock full-stack roles
4. Apply to 10–15 roles per week on Internshala, LinkedIn, and Naukri
5. Highlight the **IIT Delhi CEO Challenge** prominently — a rare differentiator
6. Practice LeetCode Easy/Medium weekly to stay sharp for DSA rounds
