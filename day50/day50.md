# Day 50: Defend Your Experience Showcase

## 📌 Overview
For Day 50 of the #60DayClaudeChallenge, I built **Defend Your Experience**, an interactive interview preparation simulator using Claude. The application extracts claims from uploaded resumes or portfolios and subjects the user to adaptive, rigorous follow-up questions to sharpen their interview readiness.

---

## 🚀 Key Features & Architectural Learnings

1. **Adaptive Question Engine**
   - Parses resume/portfolio content to generate context-aware interview questions tailored directly to the candidate's claims.

2. **Claim & Experience Validation**
   - Dynamically challenges technical claims and project descriptions with realistic, high-friction follow-ups to test real-world mastery.

3. **Automated Defense Report**
   - Evaluates response clarity, confidence levels, and technical accuracy, producing a detailed report with targeted areas for improvement.

4. **Production-Grade UI/UX**
   - A modern, responsive HTML interface engineered for real-time interview simulations and smooth user interaction.
