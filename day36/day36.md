# Day 36: Cognitive Pattern Explorer 🧠

Built a complete, interactive educational HTML application using Claude to analyze thinking styles and decision-making processes. 

### What This App Does:
* **Thinking Styles:** Explores how we naturally process information under different mindsets.
* **Dual-Mode Interactive Learning:** Features both "Calm Mode" and "Stress Mode" to evaluate everyday scenarios, drag-and-drop activities, and thinking sequences.
* **Personalized Reflection Journal:** Generates visual insights and a summary profile comparing thinking tendencies across different chapters.

### Files Included:
* `cognitive_pattern_explorer.html` - The standalone interactive application.
* `day36.md` - Key learnings and screenshots of the Reflection Journal.
