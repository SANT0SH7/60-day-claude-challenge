# Day 34: Marketing Detective Interactive Simulation Application

## 🎯 Project Overview
This project is an interactive, detective-style educational web application generated using Claude. The application simulates real-world marketing scenarios where users act as investigators—analyzing campaign evidence, identifying critical marketing mistakes, and studying customer insights through a hands-on investigation board.

## 🛠️ Tech Stack & Concepts
*   **Generation Tool:** Claude AI (Low effort level prompt execution)
*   **Frontend:** HTML5 / CSS3 / JavaScript (Interactive React-style elements)
*   **Core Concepts:** Campaign Analysis, Evidence-Based Thinking, Customer Insights, Interactive UI

---

## 🧠 Key Learnings

1. **Campaign Analysis:** Learned to evaluate mock business metrics and campaign performance indicators critically rather than relying on surface-level assumptions.
2. **Evidence-Based Thinking:** Developed a systematic approach to connecting disparate clues on an investigation board before forming a final strategic verdict.
3. **Customer Insights:** Observed how shifting target audience behaviors directly influence the success or failure of digital marketing funnels.
