# Day 48: The Verdict Engine ⚖️📊

## Project Overview
For Day 48 of the challenge, I built **The Verdict Engine**, an interactive decision-support web application designed using Claude. The application anchors every metric to real, verifiable sources, featuring dynamic priority-weighting sliders, a live source-citation panel, and a detailed "How this was researched" log that transparently resolves data conflicts.

## Key Learnings
* **Citation & Grounding Logic:** Implementing strict prompt boundaries to enforce real-world data citations and flag unverified metrics.
* **Interactive Weighting UX:** Building dynamic calculation loops that update option rankings instantaneously as user priorities shift.
* **Transparent Conflict Resolution:** Exposing source discrepancies and estimation logic directly to the user to foster trust in AI decision-support systems.

## Deliverables Included
* `index.html` (The Verdict Engine interactive application)
* Screenshots of the live ranking updates, sources panel, and research log
* Sourced data report detailing criteria and evaluation metrics
* Key technical learnings summary
