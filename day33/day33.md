# Day 33 – Media Integrity Analyzer 🧠

**60 Day Claude Challenge | Media Literacy with Claude**

## What I Built
An interactive single-file HTML application that teaches media literacy through guided discovery — not memorization. Users actively analyze real-world style headlines and emotional language patterns instead of just reading about them.

## Features
- 🔍 **Headline Detective** — Analyze how headlines influence perception and attention; get a rewritten neutral version
- 😤 **Emotion Detector** — Identify fear, outrage, and urgency triggers embedded in media content
- 📊 **Media Integrity Dashboard** — Live metrics tracking your critical thinking performance
- 🎨 **Theme Toggle** — Multiple color themes for accessibility preference
- 🔄 **Randomized Scenarios** — Replay with new content each time for continued learning

## Tech Stack
- Single-file HTML (no frameworks, no backend)
- Vanilla JavaScript
- CSS custom properties for theming
- Claude AI (prompt-driven content generation)

## Key Learnings
- Headlines are engineered to trigger emotional responses *before* rational thought kicks in
- Recognizing manipulation doesn't eliminate its emotional effect — awareness must be practiced
- Interactive learning through discovery retains far better than passive reading
- Claude can generate nuanced, educational experiences entirely within a single HTML file
