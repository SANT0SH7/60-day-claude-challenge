🎮 Day 20 : Face Puzzle Game
Building an interactive browser game with Claude—image processing, camera integration, and responsive UI.
What I Built
A fully functional face puzzle game generated by Claude that:

Captures or uploads photos via browser camera/file input
Dynamically generates jigsaw puzzle pieces from images
Tracks completion time and scoring
Works seamlessly on desktop and mobile devices

Key Technologies

Frontend: HTML5 Canvas API, vanilla JavaScript
APIs: Browser Camera API, File Reader API
Image Processing: Canvas manipulation for puzzle piece generation
UI: Responsive CSS, mobile-first design

Learnings
✅ AI can generate complete, interactive browser applications from minimal specification

✅ Game mechanics (timing, scoring, state management) emerge naturally from structured prompts

✅ Image processing in the browser is surprisingly powerful for creative applications

✅ Responsive design patterns are critical for modern web experiences
