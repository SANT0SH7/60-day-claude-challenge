# Day 30: Build a Supply Chain Optimizer

## Project Overview
This directory contains the deliverables for Day 30 of the #60DayClaudeChallenge. The objective of this lab was to act as an operations leader by optimizing an existing, underperforming supply chain architecture using a custom-generated interactive React HTML simulator. 

Rather than engineering a new system from the ground up, this simulation focuses on continuous improvement and balancing complex, competing enterprise trade-offs.

## Key Learnings & Competencies
* **Optimization Strategy:** Developed methods to analyze and iteratively improve legacy operational configurations rather than rebuilding from scratch.
* **Complex Business Trade-offs:** Balanced five competing core operational pillars: cost, speed, risk, sustainability, and customer satisfaction.
* **Supply Chain Fundamentals:** Configured the intersection of sourcing, manufacturing nodes, warehouse positioning, fulfillment logistics, and inventory management strategies.
* **Enterprise UI Generation:** Leveraged Claude to structure a live telemetry optimization dashboard using React components embedded inside a standalone HTML document.
